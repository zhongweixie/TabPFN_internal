Key Insight

The looped predictor's advantage is in OOD generalization, not in-distribution performance.

On the synthetic SCM eval (same distribution as training), c3 (deep) slightly outperformed c2 (looped). But on real benchmark datasets (distribution shift from synthetic training data), the ranking completely reverses: c2 dominates, c3 collapses. Weight-sharing recurrence acts as a regularizer that prevents overfitting to the synthetic prior's distribution, enabling better transfer to real-world tabular data.

Experimental Setup

  • Training: TACO recipe (TabICL SCM synthetic prior), 80K steps, batch 32, AdamW lr=1e-4 cosine, from random init
  • Loop mechanism: Monkeypatched LayerStack.forward to run the 12-layer encoder K times with input re-injection: h ← LayerStack(h); h ← h + 0.1·h₀
  • Eval: 25 OpenML/sklearn classification datasets, 3 random train/test splits (seeds 42/123/7), N_train≤1000, N_test≤300, stratified. Metric: accuracy (primary), ROC-AUC (binary/multiclass OVR)
  • Fair comparison: Same training seed → identical synthetic data stream. Same eval protocol. Only architecture differs.

Why Synthetic Eval Was Misleading

Eval Typec1 basec2 loopc3 deepWinner
Synthetic SCM (same distribution as training)0.6220.6240.627c3
Real benchmark (OOD)0.8200.8390.495c2

The synthetic eval correctly measures in-distribution performance (where more parameters/compute = better). But for a pretrained model that will be deployed on unseen real data, OOD generalization is what matters — and weight-sharing recurrence provides exactly that through implicit regularization.

Implications

  1. Looped/recurrent predictor is the first positive result in this research program (after 4/4 FlashMemory NO-GOs + frozen-model failures).
  2. The mechanism is regularization through weight-sharing, not "iterative reasoning" — tabular ICL remains single-pass, but sharing weights across depth prevents overfitting to the synthetic prior.
  3. Next steps: scale up (TACO-level batch=1024, 82M datasets), try loop_k=3/4, implement Ouro-style adaptive halting, test on full TabArena benchmark.
